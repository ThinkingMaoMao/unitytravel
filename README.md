台州府城数字场景演示

这是一个基于 Unity 制作的台州府城数字场景演示项目，包含城市地形、历史街区、代表性地标、昼夜与天气表现，以及可步行探索的任务体验。

## 演示内容

- 紫阳街历史街区
- 崇和门广场与古城门户
- 台州府城墙及山体城防节点
- 东湖、亭台、荷塘与周边景观
- 世界空间静态片区标签
- 第三人称与第一人称探索
- 地图、背包、任务点、NPC 对话和传送门
- 昼夜、天气与环境音效

## Windows 演示版

推荐使用完整压缩包进行交付：

```text
Builds/FuchengDemo_Windows64.zip
```

运行步骤：

1. 将 ZIP 完整解压到本地磁盘。
2. 保持 `FuchengDemo.exe`、`FuchengDemo_Data`、`MonoBleedingEdge` 和 `UnityPlayer.dll` 位于同一目录。
3. 双击 `FuchengDemo.exe` 启动演示。
4. 若 Windows SmartScreen 提示未知发布者，请确认文件来源后选择继续运行。

> 不要只复制 `FuchengDemo.exe`。Unity Player 需要同目录下的数据与运行时文件。

## 建议演示路线

1. 从崇和门广场开始，观察广场入口、石狮和片区标签。
2. 前往紫阳街，展示历史街区、NPC 对话和任务交互。
3. 使用地图或传送门前往东湖，展示湖面、亭台、荷塘和天气效果。
4. 前往台州府城墙，展示山体城墙、平台节点和高处视野。
5. 切换昼夜、天气和第一 / 第三人称视角，展示不同环境效果。

## 建议运行环境

- Windows 10 或 Windows 11，64 位
- 支持 DirectX 11 的独立显卡或性能较好的集成显卡
- 建议 8 GB 以上内存
- 建议预留至少 3 GB 可用磁盘空间
- 建议使用 1920 x 1080 或更高分辨率

场景包含大范围地形与较多模型，首次启动和首次进入场景可能需要稍长时间。

## Unity 项目信息

- Unity 版本：`6000.3.9f1`
- 目标平台：`StandaloneWindows64`
- 脚本后端：`Mono`
- 主场景：`Assets/Scenes/New City.unity`
- 构建输出：`Builds/FuchengDemo/FuchengDemo.exe`

重新构建时，请确认 `New City` 已启用在 Build Settings 中，然后选择 Windows 64 位 Player。正式演示包建议关闭 Development Build。

## 已知注意事项

- 项目包含 CityGen3D 和 URP 着色器，首次构建需要较长的着色器编译时间。
- Unity 构建日志中存在部分旧导入资源的 Missing Script、网格三角形和着色器兼容性警告；当前 Windows 演示版已成功构建，构建错误数为 0。
- 大范围地形和高精度资源使未压缩版本约为 1.83 GB，压缩包约为 0.77 GB。
- 演示前建议在目标电脑上完整运行一次，确认显卡驱动、分辨率、音量和输入设备正常。
